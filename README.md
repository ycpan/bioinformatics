# bioinformatics
### 生物信息学学习视角
学习生物信息学包含三个方面：
1.  了解基本的背景及生物信息学相关资料（详见`参考资料/软件比较.pdf`)
2.  了解python程序的简单实用（详见`参考资料/Python for Bioinformatics (Second Edition) [EnglishOnlineClub.com].pdf`第一章及第8章)
3.  上手处理生物信息学的具体问题(详见下述处理生物信息学的具体问题)。
